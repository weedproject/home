Welcome to WeedProject.org

Open source cannabis data. Still missing a lot, help is always appreciated :)

Dispensary Data: 
$ cd weedproject/dispensaryDB
$ git fetch origin
$ git checkout gh-pages
Strain Data: 
$ cd weedproject/strainDB
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Our Purpose

Open source tools to help developers make apps using dispensary & strain data.

Support or Contact

Want to contribute?
1) Fork it.
2) Make a commit on a new branch.
3) Send a pull request and we'll add it.

Contact love@weedproject.com for questions.
